# SRL Learning Analytics Dashboard

Development of a Self-Regulated Learning Analytics Dashboard (LAD) to support Self-Regulated Learning (SRL) in the Architecture and Modelling of Management Information Systems (AMMIS) course at KU Leuven.

This repository contains the Power BI dashboard implementation, DAX calculations, supporting datasets, and dashboard screenshots developed as part of the master's thesis research project.

---

# Project Overview

The dashboard was designed to support students throughout the main phases of Self-Regulated Learning (SRL):

- Forethought
- Performance Monitoring
- Self-Reflection

The system integrates:
- progress monitoring
- peer comparison
- adaptive feedback
- weekly learning objectives
- performance tracking
- reflective scaffolding

The dashboard was implemented in Power BI using anonymized course performance data from the edX learning platform.

# Research Context

This project was developed as part of the thesis:

"Development of a Learning Analytics Dashboard to Support Self-Regulated Learning in the Architecture and Modelling of Management Information Systems Course at KU Leuven"

The research investigates:
- SRL-informed dashboard design
- peer comparison mechanisms
- student perceptions of learning analytics
- adaptive feedback systems
- mobile vs desktop dashboard usage
